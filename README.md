# Predicting Telecom Churn using Machine Learning

## 📌 Overview
This project applies Machine Learning techniques to predict customer churn in the telecommunications industry.

The analysis focuses on understanding customer behavior and identifying the factors that contribute to customer attrition.

## 🎯 Objectives
- Predict telecom customer churn
- Analyze customer behavior patterns
- Compare multiple ML models
- Provide recommendations to reduce churn

## 📊 Dataset
Source:
- Kaggle Telco Customer Churn Dataset

Dataset details:
- 7043 customer records
- 20 features + 1 target variable

Features include:
- Contract type
- Payment method
- Internet service
- Monthly charges
- Tenure
- Customer demographics

Target Variable:
- `Churn`

## 📈 Evaluation Metrics
- Accuracy

## 🔥 Key Findings
- Customers with short tenure are more likely to churn
- High monthly charges increase churn risk
- Automatic payment methods reduce churn
- Loyal customers are less likely to leave
